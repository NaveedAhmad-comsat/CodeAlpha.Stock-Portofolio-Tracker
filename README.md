📈 Task 2 — Stock Portfolio Tracker
A CLI-based investment tracker that calculates the total value of a user's 
stock portfolio using a hardcoded price database.

**Concepts:** `dictionary` · `input/output` · `arithmetic` · `file handling`

**Features:**
- 10 real-world stocks with hardcoded prices (AAPL, TSLA, NVDA, MSFT, etc.)
- Live stock catalogue viewer via the `LIST` command
- Input validation — rejects unknown tickers, duplicates, and invalid quantities
- Portfolio summary table with per-stock share percentage
- Analytics: most/least valuable holding at a glance
- Export to `.txt` (readable report) and/or `.csv` (spreadsheet-ready)
- Auto-timestamped reports

---

## 🚀 How to Run

# Stock Tracker
python stock_tracker.py
```

No external libraries required — pure Python 3 standard library only.
